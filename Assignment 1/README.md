# Basic Data Exploration and Cleaning using Pandas

## Project Overview

This project demonstrates basic data exploration and data cleaning operations using Python and Pandas.

The dataset used for this project was `tshirts.csv` from the provided ecommerce dataset collection.

## Tasks Performed

* Loaded the CSV dataset using Pandas
* Explored dataset structure and columns
* Identified and handled missing values
* Removed highly missing columns
* Checked duplicate records
* Performed filtering and selection operations
* Converted data types for numerical analysis
* Created a derived column
* Saved the cleaned dataset as a new CSV file

## Technologies Used

* Python
* Pandas
* Jupyter Notebook
* PyCharm

## Files Included

* `tasks.ipynb` → Main Jupyter Notebook
* `cleaned_tshirts.csv` → Cleaned dataset
* `tshirts.csv` → Original dataset

## Learning Outcome

This project helped in understanding:

* Data preprocessing
* Missing value handling
* Data filtering
* Column operations
* Dataset cleaning using Pandas
