# Data Engineering Assignment 003

## Objective
Analyze Superstore sales data using:

- Subqueries
- Common Table Expressions (CTEs)
- Window Functions

## Dataset
Sample Superstore Dataset

## Tables Created

1. customers
2. orders
3. products

## Concepts Covered

### Subqueries
- Orders above average sales
- Highest sales order per customer

### CTEs
- Total sales per customer
- Above average sales customers

### Window Functions
- Customer ranking
- Row numbering
- Top 3 customers

## Final Query
Used:
- JOIN
- CTE
- Window Function

to display:

- Customer Name
- Total Sales
- Rank
